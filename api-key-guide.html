01.

เปิดเว็บ Google AI Studio

ไปที่หน้า API Keys ของ Google AI Studio โดยตรง — เป็นเว็บอย่างเป็นทางการของ Google สำหรับจัดการกุญแจ Gemini ทั้งหมด

คัดลอกลิงก์นี้แล้วเปิดในเบราว์เซอร์ Chrome / Safari / Edge:
https://aistudio.google.com/app/apikey

TIP เปิดในแท็บใหม่ ของเบราว์เซอร์ที่เปิดคู่มือนี้ไว้ จะได้ทำตามขั้นตอนพร้อมกันได้ง่าย

03.

คำถาม
ที่พบบ่อย.

คำตอบสำหรับสิ่งที่หลายคนสงสัย ก่อนสมัครและใช้งาน Gemini API key

ฟรีจริงไหม? มีคิดเงินภายหลังหรือเปล่า?
ฟรีจริง ภายใต้โควต้า Free Tier ของ Gemini API — ปัจจุบัน Gemini 2.0 Flash ให้ใช้ฟรี 1,500 requests/วัน ซึ่งเหลือเฟือสำหรับการสแกนฉลากยาในชีวิตประจำวัน Google จะไม่หักเงินอัตโนมัติ เว้นแต่คุณไป upgrade เป็น Paid Tier เอง
ต้องใส่บัตรเครดิตไหม?
ไม่ต้อง — สำหรับ Free Tier ของ Gemini API ใช้แค่ Google Account ไม่ต้องผูกบัตร ไม่ต้องเติมเงิน
คีย์มีอายุไหม? ต้องต่ออายุหรือเปล่า?
คีย์ ไม่มีวันหมดอายุ ใช้ได้ตลอด เว้นแต่คุณกดลบเอง หรือ Google ตรวจพบการใช้งานผิดปกติ (เช่น ถูกขโมยไปใช้สแปม) แนะนำสร้างคีย์ใหม่ทุก 6-12 เดือนเพื่อความปลอดภัย
ข้อมูลภาพฉลากยาของฉันจะถูกส่งไปไหน?
ส่งตรงไปที่ Google Gemini API เพื่อให้ AI วิเคราะห์ ไม่ได้ส่งไปที่อื่น

Google อาจเก็บข้อมูลที่ส่งใน Free Tier เพื่อพัฒนาโมเดล ตามนโยบาย Free Tier ของ Gemini ถ้ากังวลเรื่องความเป็นส่วนตัวมาก ควรอัปเกรดเป็น Paid Tier ซึ่งไม่เก็บข้อมูล — อ่านรายละเอียดที่ ai.google.dev/gemini-api/terms
ถ้าทำคีย์หาย/ลืม จะหาย้อนหลังได้ไหม?
ดูย้อนหลังได้เต็มคีย์ — เข้าไปที่ aistudio.google.com/app/apikey คลิกที่คีย์เดิม จะเห็นค่าเต็มอีกครั้ง — หรือสร้างคีย์ใหม่แทนได้ตลอด ฟรี
ใช้ในประเทศไทยได้ไหม?
ใช้ได้ปกติ — Gemini API เปิดให้ใช้ในไทยอย่างเป็นทางการ และอ่านภาษาไทยได้แม่นยำมาก (รองรับ 30+ ภาษา)
คีย์ของคนอื่นใช้แทนได้ไหม?
ใช้ได้ แต่ไม่แนะนำ — เพราะโควต้าจะใช้รวมกับเจ้าของคีย์ และเจ้าของจะเห็น log การใช้งาน — ทางที่ดีคือสมัครของตัวเอง ใช้เวลาแค่ 1 นาที
เกิด error "API key not valid" ทำยังไง?
เช็คตามลำดับ:
1. คัดลอกคีย์ครบถ้วน ขึ้นต้น AIzaSy หรือไม่
2. ลองเข้า aistudio.google.com/app/apikey เพื่อยืนยันว่าคีย์ยังอยู่ในรายการ
3. ตรวจสอบว่า Generative Language API ถูก enable ใน Cloud Project นั้นแล้ว (โดยปกติ AI Studio เปิดให้อัตโนมัติ)
4. ลบและสร้างคีย์ใหม่ ก็เคยช่วยได้ในบางเคส

พร้อมแล้ว?
ไปเริ่มสแกนยากันเลย
